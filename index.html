API de DondeGo

La API de DondeGo da acceso a una base de eventos y lugares de Barcelona y Madrid para desarrolladores, medios, mapas, apps urbanas, productos editoriales y partners.

En DondeGo trabajamos con contenido local pensado para la vida real de la ciudad. Publicamos conciertos, exposiciones, teatro, festivales, ferias, actividades para niños, planes de temporada y propuestas urbanas que ayudan a responder una pregunta muy simple: qué hacer hoy y qué está pasando ahora mismo.

La API nace para que ese contenido pueda reutilizarse de forma estructurada en proyectos externos. Sirve tanto para construir una agenda local como para enriquecer un mapa, un buscador, una app móvil, una newsletter, un recomendador o una integración editorial.

Acceso a la API

Si quieres usar la API de DondeGo, escríbenos a bcn@dondego.es. Revisamos cada solicitud de forma individual y compartimos acceso con proyectos y colaboradores interesados.

Qué puedes consultar

La API está pensada para trabajar con dos grandes bloques de información: eventos y lugares.

En la parte de eventos puedes consultar contenido por ciudad, fecha, categoría, proximidad geográfica o combinaciones de filtros. En la parte de lugares puedes obtener recintos, espacios culturales, puntos de interés y sitios cercanos a unas coordenadas concretas.

Casos de uso habituales

Eventos de hoy en Madrid

Un caso básico es pedir los eventos del día en Madrid, igual que en la vista pública de eventos de hoy en Madrid. 

GET /api/v1/events/?location=madrid&date=hoy

Eventos cerca de un punto del mapa

Si tu producto necesita mostrar planes próximos a la ubicación del usuario, puedes consultar eventos dentro de un radio concreto a partir de unas coordenadas. 

GET /api/v1/events/?lat=40.4168&lon=-3.7038&radius=3000&date=hoy

Ejemplo orientativo: eventos de hoy en un radio de 3 km alrededor del centro de Madrid.

Lugares en un radio determinado

También puedes obtener lugares cercanos, por ejemplo espacios culturales, salas, museos o puntos destacados dentro de una zona concreta. 

GET /api/v1/places/?lat=41.3874&lon=2.1686&radius=2000

Ejemplo orientativo: lugares en un radio de 2 km alrededor del centro de Barcelona.

Filtrado por ciudad y categoría

Otro uso típico es combinar ciudad, fecha y categoría para construir listados específicos. 

GET /api/v1/events/?location=barcelona&category=exposiciones&date=hoy

Para quién está pensada

La API de DondeGo puede ser útil para medios, apps de ocio, guías urbanas, mapas interactivos, newsletters locales, asistentes, productos de recomendación y cualquier proyecto que quiera trabajar con datos útiles sobre planes y lugares en la ciudad.

También es una buena base para equipos que necesitan una capa local bien estructurada sobre Barcelona y Madrid sin tener que construir ese dataset desde cero.

Una API pensada para producto

No queríamos publicar solo una base de datos. Queríamos ofrecer una capa reutilizable de contenido local con sentido editorial, útil para interfaces reales y fácil de integrar en productos digitales.

Por eso la API de DondeGo está orientada a consultas que suelen tener valor inmediato: qué hacer hoy, qué hay cerca de aquí, qué lugares entran en esta zona, qué eventos pertenecen a una categoría o qué contenido puede alimentar una agenda local.
